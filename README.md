# StudyFlow1
Projet 4 — Gestionnaire d’Études / Révisions
Vision
App pour les étudiants organiser planning de révisions, tâches et suivre progression.
Business model
• Abonnement étudiant, partenariats écoles, vente de modules de contenu premium.
MVP
1. Ajouter matières / sessions de révision (date, durée, objectif)
2. Check-list de sessions complétées
3. Calcul du % de progression par matière
4. Notifications locales (reminders via alertes)
Cahier des charges fonctionnel
• Formulaire d’ajout session: matière, date, durée, objectif
• Affichage calendrier simple (liste par date)
• Progress bar par matière
• Export / impression du planning
